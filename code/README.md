# Code

Contains helper scripts used during Bluetooth testing and enumeration.

## Scripts
- `ble_scan.py` - placeholder scanner entry point.
- `device_enumeration.ps1` - Windows device enumeration helper.
- `rfcomm_enumerate.ps1` - RFCOMM channel enumeration helper.
- `at_commands.ps1` - AT command probing helper.
- `advanced_packet_analysis.py` - packet-capture post-processing for A2DP/AVRCP/HFP traffic in pcapng files.
